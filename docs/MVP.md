# CreatorKit AI MVP

## In Scope

- paste a content idea or draft
- choose a platform and content type
- enter hook, caption, transcript, duration, niche, and CTA intent
- receive a score, critique, strengths, risks, and suggestions
- run a local backend and local frontend

## Out of Scope

- authentication
- payment or subscriptions
- collaboration features
- saved history
- production model training

## Acceptance Criteria

- the backend returns a response from `POST /analyze`
- the frontend submits the form and renders the response
- the app can be run locally with clear setup steps
- the notebook prototype remains preserved under `research/`
